# scoring_engine
Scoring Engine for Red/White/Blue Team Competitions

# DEV INSTRUCTIONS

1. Install Vagrant

2. Install Virtualbox

3. Fork this repo

4. Clone your fork

5. `vagrant up` then `vagrant ssh`

6. `cd /vagrant; pip install -e .`

7. `py.test tests`
